Welcome

Welcome to the ML-SPL Performance App.

This app demonstrates the performance experiments of ML-SPL (Machine Learning - Search Processing Language), which encompasses the commands and algorithms underneath the Machine Learning Toolkit App.

Using this dashboard, you can browse the results of performance testing of ML-SPL. For each algorithm implemented in ML-SPL, we measure running time, CPU utilization, memory utilization, and disk activity when fitting models on up to 1,000,000 search results, and applying models on up to 10,000,000 search results, each with up to 50 fields.

To learn more about the Machine Learning Toolkit App, please refer to the User Guide of Machine Learning Toolkit.
